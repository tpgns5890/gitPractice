# GIT bash 명령어

- 폴더 오른쪽 클릭 > git bash 실행

>git init

>git config --global user.email "***@***"

>git config --global user.name "****"

>git remote add origin //올릴 저장소 주소
(https://github.com/tpgns5890/gitPractice.git)

>git remote -v //주소 비교

>git add * (모든 파일) //파일 올리기

>git commit -m 'first' //커밋 달기

>git push origin master //푸쉬

*https://backlog.com/git-tutorial/kr/ 사용법
